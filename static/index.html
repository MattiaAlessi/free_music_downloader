⚙️ Come funziona

Questa modalità usa spotdl, un tool open-source che:

  • 🔑 Non richiede account Spotify o API keys
  • 🔍 Legge la tracklist dalla pagina Spotify pubblica
  • ▶ Scarica l'audio da YouTube al massimo della qualità
  • 🏷️ Aggiunge automaticamente tag ID3 (artista, album, copertina)
  • 📦 Si installa automaticamente al primo utilizzo
Dipendenza: spotdl — status visibile nella sidebar. Se non rilevato, viene installato automaticamente all'avvio del download.
